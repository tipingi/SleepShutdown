# SleepShutdown.WinForms

지정된 시간 후에 컴퓨터를 종료하거나, 최대 절전 모드로 전환하거나, 모니터를 끄도록 예약하는 간단한 Windows Forms 애플리케이션입니다.

## 주요 기능

- **동작 예약**: 지정된 시간(분) 후에 컴퓨터가 종료되거나, 최대 절전 모드로 전환되거나, 모니터가 꺼지도록 예약합니다.
- **타이머 제어**: 타이머를 시작, 일시정지, 재개, 취소할 수 있습니다.
- **카운트다운**: 예약된 동작이 실행될 때까지 남은 시간을 표시합니다.
- **시스템 트레이 연동**: 애플리케이션을 시스템 트레이로 최소화하여 백그라운드에서 실행하고 제어할 수 있습니다.
- **종료 중단**: 예약된 시스템 종료를 취소하는 기능을 제공합니다.

## 사용 방법

1. 애플리케이션을 실행합니다.
2. 드롭다운 메뉴에서 원하는 동작(종료, 최대 절전 모드, 모니터 끄기)을 선택합니다.
3. 시간을 분 단위로 설정합니다.
4. "시작" 버튼을 클릭합니다.
5. 타이머가 시작되며, 일시정지/재개 또는 취소할 수 있습니다.
6. 애플리케이션 창을 닫으면 시스템 트레이로 최소화되어 백그라운드에서 계속 실행됩니다.

## 빌드 방법

1. 저장소를 복제합니다.
2. Visual Studio에서 `SleepShutdown.WinForms.sln` 파일을 엽니다.
3. 솔루션을 빌드합니다(Ctrl+Shift+B).
4. 실행 파일은 `src\bin\Debug` 또는 `src\bin\Release` 디렉터리에서 찾을 수 있습니다.
